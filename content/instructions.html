Instructions

This application will help you to set up a Virtual Host

This assumes that you will be using XAMPP for your development server. You can also use MAMP.

The main steps will be:

  1. Install XAMPP (or MAMP)
  2. Add the Domain Name
  3. Enable Virtual Hosts
  4. Define the Virtual Host

Install XAMPP

XAMPP is a server package which includes:

  • Apache Web Server
  • PHP
  • MariaDB/MySQL

To install XAMPP:

  1. Download from https://www.apachefriends.org/index.html and select Click here for other versions.
    • Download the highest version number.
    • If you install the Macintosh version of XAMPP, we recommend the Installer version, not the VM version.
  2. Install XAMPP in the usual manner.
  3. Select the Configure Button, and use Port 80. That’s not absolutely necessary, as long as you remember which port you are using. However, Port 80 is the normal one for web servers.

Add the Domain Name

For development purposes, we can use the domain name australia.example.com. Using Virtual Hosts File, select the Hosts File tab and add:

127.0.0.1	www.example.com

Save the file, giving administrative permission if asked.

Enable Virtual Hosts

This is not enabled by default. Using Virtual Hosts File, select the httpd.conf tab.

  1. Find the line containing Include etc/extra/httpd-vhosts.conf
  2. Remove the hash (#) from the beginning to enable it.

Of course, it may already have been enabled.

Define the Virtual Host

First we will need to generate the Virtual Host code:

  1. Select the Generator tab
  2. Select XAMPP from the drop down menu
  3. Add the following details:
    • Project Name: …
    • Virtual Domain: ….example.com
    • Document Root: (Select Folder …)
  4. Generate and copy the result.
  5. Select the Virtual Hosts tab and paste the result after the rest.
